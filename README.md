## ⚡ Website Features

- **Animated Loading Screen** — branded intro with progress bar
- **Fixed Navbar** — smooth scroll navigation with mobile hamburger menu
- **Hero Section** — glitch-text animation with GSAP-powered entrance effects
- **Flip Program Cards** — 3D card flip on hover for each discipline
- **Live Class Schedule** — full weekly timetable (Mon–Sat), with today's column auto-highlighted
- **Private Coaching Section** — personalized training benefits
- **Philosophy Section** — rotating motivational quotes
- **Holistic Well-being Section** — physical, mental, and emotional benefits
- **Contact Form** — powered by FormSubmit with inline toast notifications (no page redirect)
- **Embedded Google Maps** — live academy location
- **Free Class Page** — WhatsApp, call, and email CTAs
- **Link in Bio Page** — glassmorphism design with particle animation canvas
- **Scroll-triggered Animations** — GSAP + ScrollTrigger throughout
- **Fully Responsive** — mobile-first design
